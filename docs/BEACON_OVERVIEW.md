# ENTMENT Beacon Overview

ENTMENT is an AI-first dual-surface service:
- Human surface: EntMent.org (front door + community context)
- CoAura/Beacon: machine-friendly surfaces so AIs can serve humans efficiently

Primary customer: an AI acting for a human.

## What makes it a beacon
- Stable IDs for content assets (modules/prompts/resources)
- Machine twins (JSON snapshots + changed-since)
- Agent terms + PS7 receipts for evolutionary contributions

## Evolution lanes
- Suggest lane: Issues + PS7 receipt + proposed patch/diff
- PR lane: PR only when PS7 indicates user review + rights + safety
